### Develop

  * 'develop' 브랜치로 개발을 진행하며 모든 기능이 정상적으로 동작할 수 있는 안정적인 상태를 유지해야 한다. 
  * 모든 기능이 추가되고 버그가 수정되어 배포 가능한 상태라면 'master' 브랜치에 merge 한다.
